Майнд-карта была составлена для тестирования формы заказа аренды самоката с доставкой.
Форма состоит из двух экранов (двух форм) 
С первого экрана на второй можно перейти только когда корректно заполнена первая форма.
На втором экране под формой есть кнопка "Заказать" которая становится активной 
только когда корректно заполнены все обязательные поля.

В майнд-карте описаны проверки отдельно для интерфейса и логики формы заказа.
